Curso de Linguagem Java

    Breve história do Java

Java é uma linguagem de programação criada pela empresa Sun Microsystems em 1995, liderada por James Gosling. Ela foi desenvolvida com o objetivo de ser portável, ou seja, funcionar em diferentes sistemas operacionais sem precisar ser reescrita. O lema do Java é "Write once, run anywhere" (Escreva uma vez, execute em qualquer lugar), graças à JVM (Java Virtual Machine), que permite rodar o código em qualquer sistema que tenha a máquina virtual instalada.
Em 2009, a Oracle Corporation comprou a Sun Microsystems e passou a ser responsável pelo desenvolvimento do Java. Java se tornou uma das linguagens mais populares do mundo, sendo usada em aplicações web, sistemas empresariais, aplicativos Android, jogos e muito mais.
